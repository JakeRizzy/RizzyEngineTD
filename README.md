# RizzyEngineTD

The development of a 2D game engine utilizing C++20 / SFML 2.6.1 / ImGui 1.89.9 / ImGui-SFML 2.6 (RizzyEngine) and subsequent playable demo (Escape Room) to serve as the foundation for my 2D game development and design projects.

Branching: main is for fully functional, stable, and tested code while development is for code being actively implemented and/or tested.

URL to Confluence Wiki for full documentation suite and Jira board for the project: https://jakerizzy.atlassian.net/wiki/spaces/REDW/overview
